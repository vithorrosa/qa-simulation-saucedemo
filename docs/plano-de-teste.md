# Plano de Teste — SauceDemo

## Objetivo

Validar os principais fluxos da aplicação SauceDemo, simulando a rotina real de um QA Júnior em um projeto de e-commerce.

## Escopo

Serão testadas as seguintes funcionalidades:

- Login
- Listagem de produtos
- Carrinho
- Checkout
- Logout

## Fora de escopo

Nesta primeira fase, não serão realizados:

- Testes de performance
- Testes de segurança avançados
- Testes mobile
- Testes de API

## Tipos de teste

- Teste funcional manual
- Teste exploratório
- Teste de regressão
- Teste automatizado E2E com Cypress

## Ambiente de teste

- Aplicação: SauceDemo
- Navegador: Chrome
- Sistema operacional: Windows
- Ferramentas: GitHub, Markdown, Cypress

## Critérios de entrada

- Aplicação acessível
- Credenciais de teste disponíveis
- Navegador instalado
- Repositório configurado

## Critérios de saída

- Casos de teste documentados
- Testes executados
- Bugs registrados, caso encontrados
- Evidências organizadas
- Cenários críticos automatizados
